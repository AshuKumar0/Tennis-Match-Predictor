# 🎾 Tennis Match Prediction

A Machine Learning project that predicts the winner of ATP tennis matches using historical player statistics and Elo ratings.

## 📌 Features

- Historical ATP match data (2010–2024)
- Data preprocessing and feature engineering
- Elo rating integration
- Logistic Regression model for prediction
- Match prediction scripts
